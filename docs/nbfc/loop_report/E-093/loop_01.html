E-093 — Score Override

Loop 01 — pass (5/5 ACs) — branch nbfc/E-093

Summary

NBFC Risk Manager can override a borrower's computed credit score with a documented reason. The override is logged to audit_logs with action score.override.created but does NOT mutate the computed score. New table nbfc_score_overrides holds the append-only history; a new override flips any prior is_active row to false for the same (loan_application_id, score_type).
